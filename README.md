### üõ†Ô∏è Initial Setup

When you run the software for the first time, you will see a welcome screen. Follow these steps:

1. **Configure Your Network Devices**

2. **Select Metrics to Monitor**  
   Choose the metrics you want to gather. This may include bandwidth usage, CPU load, and inventory data.

   Set a schedule for how often you want your data to be collected, whether hourly, daily, or weekly.
